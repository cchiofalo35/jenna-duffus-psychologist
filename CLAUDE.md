# Jenna Duffus - Psychologist Website

This is the website for Jenna Duffus, a psychologist. It is a static HTML site hosted on GitHub Pages at jennaduffus.com.

## Site Structure

- index.html — Main landing/home page
- about.html — About Jenna page
- services.html — Services offered
- faq.html — Frequently asked questions
- resources.html — Resources for clients
- workshop.html — Workshop information
- pdfs/ — Downloadable PDF resources
- CNAME — Custom domain config (jennaduffus.com)

## Guidelines for Making Changes

1. Only change what is requested — Do not redesign or restructure anything unless specifically asked.
2. Preserve the existing design — Keep all styling, colors, fonts, and layout consistent.
3. Be careful with HTML — This is a static site with no build step. All changes are directly to HTML files.
4. Keep it professional — This is a healthcare professional's website.
5. When in doubt, ask — If a request is ambiguous, comment on the issue asking for clarification.
